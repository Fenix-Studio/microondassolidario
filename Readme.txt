Microondas Solidário

O Microondas Solidário é uma iniciativa social que une sustentabilidade ambiental e responsabilidade social. O projeto foca no reparo de fornos micro-ondas descartados ou danificados, transformando o que seria lixo eletrônico em utilidade doméstica para famílias em situação de vulnerabilidade.

Sobre o Projeto

Este projeto nasceu como uma Atividade Extensionista acadêmica, pautada nos Objetivos de Desenvolvimento Sustentável (ODS) da ONU. Ele reflete a transição de carreira da sua idealizadora, unindo o conhecimento técnico prático com as melhores práticas de desenvolvimento web e segurança cibernética.

    "O que antes era sucata, nas minhas mãos vira uma ferramenta de dignidade."

Tecnologias Utilizadas

Para a construção da plataforma, foram aplicados conceitos modernos de Front-End aprendidos durante o curso de Análise e Desenvolvimento de Sistemas:

    HTML5 Semântico: Para acessibilidade e melhor ranqueamento (SEO).

    CSS3 (Flexbox & Grid): Design responsivo e estilização personalizada (estilo cartoon).

    Google Calendar API: Integração para agendamento de bazares e coletas.

    FormSubmit: Gerenciamento de leads e solicitações de doação via e-mail.

    Segurança & LGPD: Implementação de camadas de privacidade e consentimento de dados sensíveis.

Segurança e Privacidade

Como estudante de Segurança Cibernética, o projeto foi desenhado respeitando a LGPD (Lei 13.709/2018):

    Formulário com cláusula de consentimento explícito.

    Opção de entrega em ponto fixo para proteger o endereço residencial do doador.

    Tratamento mínimo de dados apenas para fins logísticos.
